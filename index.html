Code

Mo'Astro

Derived from MongoDB astronomy, Mo'Astro is a framework for building observational astronomy pipelines around the schema-less MongoDB data store. With Mo'Astro, managing the metadata and pipeline state for thousands of observations becomes a bit easier. Mo'Astro includes hooks into the Astromatic pipeline software.

Ghostmap

Delaunay and Voronoi tessellations for astronomy.

Daopilot

Python drivers for the DAOPHOT/ALLSTAR suite of photometry software by Peter Stetson.

pySPS

Python interface to the Flexible Stellar Population Synthesis Package by Conroy, Gunn and White.

MapScale

Embarassingly parallel processing with the ZeroMQ message-passing framework. Replaces for multiprocessing's Pool.map().
